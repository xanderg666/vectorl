Pasos para usar el ewallet.p12
1. Accede a Database Tools
Ve a la consola de OCI y navega a: Developer Services → Database Tools → Connections​

2. Crea una nueva conexión
Haz clic en "Create Connection"​

3. Configura la conexión básica
Name: Dale un nombre descriptivo (por ejemplo, "AI-Agent-ADB-Connection")

Compartment: Selecciona el compartimento donde está tu Autonomous Database

Select Database: Elige tu Autonomous Database de la lista​

4. Configura las credenciales de usuario
User Name: Usuario de base de datos (por ejemplo, ADMIN)

User Password Secret: Aquí debes crear o seleccionar un secreto en OCI Vault con la contraseña del usuario de base de datos​

Si necesitas crear el secreto:

Haz clic en "Create Password Secret"

Dale un nombre al secreto

Selecciona el Vault y la Encryption Key

Ingresa la contraseña del usuario de base de datos​

5. Configura el Wallet (Paso crítico)
En la sección "Secure Connection":

Wallet Format: Selecciona "PKCS#12" (no SSO wallet)​

Haz clic en "Create Key Store Content Secret"​

6. Sube el archivo ewallet.p12
En el diálogo que aparece:

Name: Dale un nombre (por ejemplo, "adb-wallet-secret")

Vault: Selecciona tu Vault

Encryption Key: Selecciona la clave de encriptación

Upload auto login wallet: Aquí es donde subes tu archivo ewallet.p12​

Puedes arrastrarlo directamente al campo "Wallet"

O hacer clic en "Select a file" y navegar hasta donde guardaste el ewallet.p12​

Necesitarás ingresar la contraseña del wallet que estableciste cuando descargaste las credenciales de la Autonomous Database​

7. Finaliza la creación
Revisa que toda la configuración esté correcta

Haz clic en "Create" para crear la conexión​

8. Verifica la conexión
Una vez creada, ve a Actions → Launch SQL para verificar que la conexión funcione correctamente antes de usarla en AI Agents​
